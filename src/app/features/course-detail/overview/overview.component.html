Course Outline

Dalam kursus ini, kalian akan diajarkan Typescript dari dasar hingga fitur-fitur penting dari Typescript sampai di titik kalian dapat mengimplementasikan Typescript ke dalam proyek apapun yang kalian kerjakan.
Yang akan anda dapatkan setelah mengikutin course ini adalah:
  • Kemampuan pemrograman typescript
  • Pemahaman mengenai fungsi dari typescript
  • Pemahaman mengenai tata kerja typescript secara teknikal
  • Kemampuan membuat simple application dengan typescript
